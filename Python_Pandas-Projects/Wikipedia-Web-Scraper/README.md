# Wikipedia Web Scraper Project

## Overview
This project demonstrates how to extract and clean unstructured data from Wikipedia using Python. It focuses on parsing complex HTML tables and transforming them into a structured format for analysis.

## Features
* **Web Scraping:** Utilizes BeautifulSoup to locate and extract specific table data.
* **Data Cleaning:** Employs Pandas to handle missing values, correct data types, and reformat columns.
* **Analysis Ready:** Outputs a clean CSV file suitable for visualization or further exploratory data analysis.

## Tools Used
* **Python**
* **BeautifulSoup4**
* **Pandas**
* **Requests**
