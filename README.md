# Data Analyst Portfolio Projects
This repository contains a collection of end-to-end data analysis projects. Each folder includes the raw data, processing scripts (Python/Jupyter), and the final insights.

Projects Included:
Amazon Web Scraper: Automated price tracking using BeautifulSoup and Selenium.

Wikipedia Data Pipeline: Real-time data extraction and cleaning with Pandas.

Power BI Dashboards: Links to interactive visualizations.

Feel free to explore the code and reach out if you have any questions about the methodologies used!
