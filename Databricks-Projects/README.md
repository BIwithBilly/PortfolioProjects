# Databricks Data Analysis & Dashboards

## Overview
This folder contains a collection of big data analysis projects and interactive dashboards developed within the Databricks environment using SQL and PySpark.

## Key Projects
* **Transaction Dashboard:** Analysis of business operational data and financial metrics.
* **Emissions Analysis:** Environmental impact tracking and data visualization.
* **NYC Taxi Data:** Large-scale geospatial and temporal data processing.

## Tools & Skills
* **Languages:** SQL, PySpark
* **Environment:** Databricks Unified Analytics Platform
* **Visualization:** Databricks Dashboards, Genie Spaces
